VUE Converter

Tool to convert VUE concept maps to various document formats (HTML, ODT, PDF and Markdown). See application in use at http://appstandards.de

What do I need a VUE Converter for?

The VUE Converter takes an old tool - the Visual Understanding Environment - and adds a new purpose to it. While still being able to draw concept maps and visualize your understanding with the VUE Converter you can get more value from node and link notes.

As you can see in the example map above the pencils next to node and link labels indicate further notes embedded. The original tool shows them when the mouse hovers over the icons. This makes it difficult to read all the information embedded.

New qualities of extracting notes

As the export functions of VUE are not very good, I decided to build the VUE Converter. It runs as a web application that you can upload VUE files and packages to. Subsequently it offers the extraction of the map's notes as PDF, HTML, ODT and Markdown.
