VUE Converter

This tool converts a VUE concept map to HTML, Markdown, PDF and Libre-/OpenOffice ODT. Compared to the native export and print function of VUE it nicely outputs notes inside nodes and links.
Give it a try!

This first version will not:

  • show your image resources. Instead it shows a cat instead of the real image
  • show a screenshot of your map. This will be possible soon.

Features to come:

  • Upload of a zip file that contains image resources and a screenshot of the map.
  • Build a references list from sources in the notes.

Want more? Request features on the Github Issue page.
